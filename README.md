# Helperbird website [![Build and Deploy](https://github.com/Helperbird/website/actions/workflows/build.yml/badge.svg)](https://github.com/Helperbird/website/actions/workflows/build.yml)

## Getting started

### Install assets

```bash
npm install
```

### Build files

```bash
npm run build
```

### Run locally

```bash
npm run dev
```
